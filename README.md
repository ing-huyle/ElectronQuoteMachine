# ElectronQuoteMachine

An Electron application with React, Node.js and TypeScript.

[Watch demo -->](https://github.com/user-attachments/assets/362c56e3-1352-448e-a0a3-96d481a0151f)

## Project Setup

### Install

```bash
$ npm install
```

### Run

```bash
$ npm run dev
```

### Optional
The app also uses a PostreSQL database but it is not needed to run the app.
